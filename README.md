# 📦 GestiónPro

Aplicación web para gestionar tu negocio: inventario, costos, precios de venta y mucho más. **No requiere instalación ni servidor** — funciona solo con tu navegador.

## ✨ Funcionalidades

- **📋 Inventario** — Productos con costo, precio, stock, proveedor y stock mínimo
- **💡 Precio sugerido** — Calcula automáticamente el precio de venta según el margen
- **💰 Ventas** — Registra ventas y descuenta stock automáticamente
- **📊 Dashboard** — Gráficas de ventas, productos top y categorías
- **🧮 Calculadora** — Costos con envío, empaque, impuestos y comisiones + simulador de ganancias (ROI)
- **🔔 Alertas** — Stock bajo, productos a pérdida, sin movimiento
- **📈 Reportes** — Rentabilidad por producto, categoría y resumen mensual
- **📜 Historial** — Registro de todos los movimientos
- **🌙 Modo oscuro**
- **📥 Exportar/Importar CSV** y **💾 Backup/Restauración** (JSON)

## 🚀 Cómo usar

1. Descarga los archivos `index.html`, `styles.css` y `app.js` en una misma carpeta.
2. Haz doble clic en `index.html`.
3. ¡Listo! Se abre en tu navegador.

Tus datos se guardan localmente en el navegador (localStorage), así que no se pierden al cerrar.

## 🛠️ Tecnologías

- HTML, CSS y JavaScript puro (sin frameworks)
- [Chart.js](https://www.chartjs.org/) para las gráficas
